# File Allocation Simulation in C

This project demonstrates file allocation techniques used in Operating Systems.

## 🔧 Features
- Sequential Allocation
- Linked Allocation
- Indexed Allocation
- Disk block visualization

## ▶️ How to Run

1. Compile:
   gcc index.c -o index

2. Run:
   ./index

## 💡 Concepts Used
- File Systems
- Memory Management
- Disk Allocation Strategies
